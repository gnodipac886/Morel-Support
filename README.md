## How the scoring works

Each grid cell gets an independent score (0вҖ“1) for each enabled layer. Those scores are combined into a single probability using a weighted average:

```
P = ОЈ(score_i Г— weight_i) / ОЈ(weight_i)
```

**Default weights:**

## Layer Reference

### iNaturalist Sightings

Fetches all *Morchella* genus observations from the iNaturalist API within the viewport. Observations with obscured locations or geoprivacy set are excluded. Each sighting is scored by:

- **Distance** from cell center (max 25 miles, power curve)
- **Seasonal alignment** вҖ” sightings near the same day-of-year as the target date score higher
- **Recency** вҖ” older observations are down-weighted by 8% per year

### Precipitation

Pulls recent rainfall from three sources and blends them per grid cell:

- **Open-Meteo Archive API** вҖ” precipitation + snowfall + 0вҖ“7 cm soil temperature
- **NASA Daymet** вҖ” independent daily gridded precipitation (CONUS + southern Canada)
- **NOAA ACIS / PRISM** вҖ” 4 km gridded precipitation from the PRISM Climate Group

Scoring curve:

| Rainfall (past window) | Score |
|---|---|
| < 0.2 in | 0.05 вҖ” too dry |
| 0.5вҖ“1.0 in | 0.60 вҖ” promising |
| 1.0вҖ“2.0 in | 0.85 вҖ” good |
| 1.0вҖ“3.5 in | 1.00 вҖ” ideal |
| > 6.0 in | 0.45 вҖ” waterlogged |

Soil temperature (0вҖ“7 cm depth) is blended at 45% weight alongside precipitation:

| Soil temp | Score |
|---|---|
| 50вҖ“60 В°F | 1.00 вҖ” optimal |
| 42вҖ“50 В°F | 0.75 вҖ” early season |
| 68вҖ“78 В°F | 0.25 вҖ” too warm |

### Fire Perimeters

Pulls historical and current fire perimeters from:

- **NIFC WFIGS** вҖ” interagency fire perimeters (current + year-to-date + historical)
- **USFS EDW** вҖ” MTBS mapped fire perimeters (1984вҖ“present)
- **CAL FIRE / FRAP** вҖ” California-specific per-year perimeters + live incident CSV
- **NASA FIRMS** вҖ” MODIS satellite hotspot detections (requires `FIRMS_MAP_KEY`)

Each fire is scored by age (peak at 1 year post-burn). Nearby fires not directly overlapping a cell still contribute a proximity score, falling to zero at 20 miles from the perimeter edge.

### Host Trees

Species scored by known morel association:

| Species | Score |
|---|---|
| Ash | 0.90 |
| Elm | 0.85 |
| Tulip poplar | 0.80 |
| Cottonwood | 0.70 |
| Douglas fir* | 0.70 |
| Oak | 0.65 |
| White fir* | 0.65 |
| Sycamore | 0.65 |
| Pine* | 0.60 |

\* Burn-zone species вҖ” receive a 1.45Г— boost when near active fire perimeters.

Sources: **GBIF** species occurrences, **LANDFIRE EVT** vegetation types, **USFS TreeMap 2016**, **NLCD 2019** land cover.

| < 300 ft | 0.30 вҖ” valley floor |
| > 8,000 ft | 0.10 вҖ” too high |

### Seasonality

A pure calculation вҖ” no network call required. Estimates peak morel day-of-year based on latitude:

```
peak_day вүҲ 70 + (latitude вҲ’ 30) Г— 2.5

Examples:
  30В°N вҶ’ ~day 70  (Mar 11)
  38В°N вҶ’ ~day 90  (Mar 31)
  45В°N вҶ’ ~day 107 (Apr 17)
  50В°N вҶ’ ~day 120 (Apr 30)
```
